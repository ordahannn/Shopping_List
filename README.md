# Shopping List App

**Shopping List App** is a cross-platform mobile application built with **React Native** and **Expo Router**. It helps users manage their grocery shopping efficiently through a sleek, responsive interface and persistent storage. This project was developed for educational purposes and showcases a modular, maintainable mobile development architecture.

## Features

- Add, update, and remove shopping items  
- Organize items with intuitive UI  
- Persistent storage using `AsyncStorage`  
- Responsive layout for both Android and iOS  
- Smooth navigation with `React Navigation`  

## Technologies Used

- **React Native**
- **Expo**  
- **Expo Router**  
- **TypeScript**  
- **AsyncStorage** for data persistence  
- **React Navigation**
