# Shopping List

**Shopping List** is a mobile application built with **React Native** and **Expo Router** for managing a personal grocery list. It allows users to add, edit, and delete shopping items, with data stored locally for offline use.  
The app uses a clean and responsive UI.

---

## Features

- Add new items to the shopping list.
- Edit or delete existing items.
- Suggestions for common grocery items.
- Local data storage using **AsyncStorage** (works offline).
- Light and dark theme support using `useThemeColor`.
